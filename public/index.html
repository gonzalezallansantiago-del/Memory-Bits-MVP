Memory Bits

Un lugar para dejar algo que no quieres que se pierda.

1. Accede con tu código.
No es una llave. Es una decisión.

2. Dale un título.
Nombrar es elegir qué permanece.

3. Escribe.
Lo que fue, lo que es. O lo que nunca dijiste.

4. Compártelo si debe viajar.
Como imagen, símbolo o un QR esperando ser encontrado.

5. Este es tu espacio.
Aquí no se explica. Aquí dejas un legado.
