Memory Bits

Un lugar para dejar algo que no quieres que se pierda.

1. Accede con tu código.
Todo empieza aquí.

2. Dale un título.
Nombrar es elegir qué permanece.

3. Escribe.
Lo que sucedió, lo que ahora es. O lo que nunca dijiste.

4. Compártelo si debe ser encontrado.
Como imagen o como un QR esperando a alguien.

5. Este es tu espacio.
Aquí dejas un legado.

6. Puedes buscar entre las memorias del mundo.
La búsqueda funciona solo por el título de la Memory Bit.

Importante:
Como en cualquier lugar de internet, no compartas datos personales ni sensibles. Memory Bits guarda únicamente lo que tú decides dejar.
